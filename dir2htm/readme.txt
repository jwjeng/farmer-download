          目錄轉HTML格式程式 Directory to Html 使用步驟


    本程式可將我們蒐集的大量資料檔案如圖形檔、音效檔或文件等等，自動轉
為html格式，以便於利用Netscape或IE等WWW瀏覽程式使用。除可用於個人資料
整理與收藏外，更利於網站管理人員建置大量資料庫時使用，可大大減少編寫網
頁的時間與精力。歡迎多加利用，希望能提供大家更好的網路資源服務。

【預備步驟】

1.建立目錄清單檔：

      進入 DOS 環境下，切換至你要轉換的目錄位置，打入 DIR /s > LIST
  以建立目錄清單。其中 LIST 為所得之清單檔，亦可用別的檔名。

2.編修目錄清單檔：

      請使用文書編輯軟體如 PE2、漢書等進行必要之編修：

(1) 將清單檔中不欲在網頁上出現的檔案去除。
        注意僅去除檔案即可，其餘的資料如目錄位置、檔案數等不要刪除；
    空白行及 [.]、[..] 等會自動跳過，不須刪除。

(2) 設定目錄樹名稱及附加文字
        你可以為整個目錄另外設定一個名稱，如[軟體下載區]或[東方美女專輯]
    等等，程式會以此名稱做為所有子目錄之根目錄名稱。
        此名稱請放在目錄清單檔的第一行。如未設定，會自動以 . 代替，可在
    程式執行中重新設定。

(3) 設定附加文字
        你可以在每個網頁下方寫入固定的說明文句，如版權聲明或注意事項等
    等。此附加文字可在目錄清單檔第二行起設定，最多10行。第一行為前述之
    目錄樹名稱，即使沒有設定也請空下來，以免搞亂。
        注意，清單檔最開頭自動出現之之   Volume in drive C is xxxx
    請務必留著，這一句是用以判定附加文字結束的關鍵，刪除就玩不下去了。
        網頁更新日期程式會自動產生，不須另外在這裡設定。

(4) 加入檔案說明
        本程式會將檔案名稱轉換為網頁中之超鏈結按鍵名稱。如要改用其他的
    名稱，請在檔案資訊之日期時間後面加上你想加入的鏈結字句即可。例如：

      AI015    JPG       155,445  06-13-96  18:27 飯島愛#15

        如無說明，則自動使用檔名作為鏈結按鈕，不須另行設定。

        如果你的檔案是超過8字以上的長檔名，在DOS下會變成像MARIYA~1.JPG
    這樣的檔案名稱。如果你想使用Win 95下的長檔名，可在 Windows 95/98 之
    MS DOS 多工視窗(非前面所說的關閉後進入之 DOS 環境)中另行建立一個目
    錄清單檔，此時該清單檔後面便會自動加入各檔案的實際檔案名稱。如此只
    要再將這些長檔名合併到前面所建立的清單檔後面即可。
        注意，不可直接使用這種具有長檔名的目錄清單檔，因為本程式係在DOS
    下執行，無法處理具有長檔名的目錄位置，故需另行轉接合併。
        另外，DOS對於某些特殊怪異的目錄名稱無法接受，因此在建立目錄名稱
    時，請盡量簡單明瞭，再利用附加之說明文字設定即可。

3.如果上述預備步驟均已完成，只要在欲轉換之目錄位置執行DIR2HTM程式即可。
  此程式可放置於path支援之位置，如C:\windows等，以便隨傳隨到。

【程式執行說明】

1.請輸入清單檔名：以前述為例即為 LIST
2.輸入Homepage檔名：
    此檔名將自動出現在以下各子目錄中，如原先已有相同檔案者將會被蓋掉。
    內定為 list.htm，以與一般慣用之 index.htm 區分。
3.設定目錄樹的名稱：會先將預設之目前目錄樹的名稱列出，不改按ENTER保留。
4.目錄區是否加入表格線?
      為避免畫面太複雜，內定為不要。但若說明文字太長，則加上比較妥當。
5.檔名需改為小寫?
      如果你只在硬碟中觀看這些檔案，自無此需要。但若要放上網站與大家分
  享，由於部份網路主機會將檔名改為小寫，如此則需先將各網頁之鏈結檔名改
  一下，以免在硬碟看明明好好的，放上網站後就不能用了。
6.目錄表現型式：
      有兩種，內定為第一種。以上述範例說明(下方橫線表網頁中之超鏈結鍵)：
(1) 檔名 + 說明
      AI015.JPG       155,445  06-13-96  18:27 飯島愛#15
      ---------
(2) 無檔名，直接以說明做為鏈結
      飯島愛#15       155,445  06-13-96  18:27
      ---------
